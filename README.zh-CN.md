# Mermaid Editor

[繁體中文](README.zh-TW.md) | **简体中文** | [English](README.md)

桌面版 Mermaid 图表编辑器，支持可视化拖拽式编辑、连线绘制、网格吸附、自定义节点颜色。

## 功能

### 代码模式
- **Mermaid 渲染** — 输入 Mermaid 语法即时渲染图表（`Ctrl+Enter`）
- **拖拽 + 网格吸附** — 自由拖拽节点，自动吸附至可配置的网格
- **自定义颜色** — 点击节点可自定义背景色、边框色、文字色
- **边线跟随** — 拖拽节点时连接线平滑跟随移动

### 可视化模式
- **元素工具箱** — 拖拽流程图元素（处理、判断、终端、输入/输出、子程序、数据库）到画布
- **Shift+拖拽连线** — 按住 Shift 从一个节点拖拽到另一个节点即可建立连线
- **点击选取连线** — 点击画布上的连线即可选取（加宽点击范围方便选取）
- **双击编辑标签** — 双击连线在点击位置弹出浮动输入框编辑标签
- **Delete 键删除** — 删除选取的节点或连线
- **双击重命名** — 在画布上双击节点可直接修改名称
- **实时代码同步** — 可视化操作实时更新 Mermaid 代码
- **方向控制** — 切换上→下或左→右布局
- **节点 & 连线列表** — 左侧面板显示所有节点与连线，支持 inline 编辑和删除

### 通用功能
- **多标签页** — 同时编辑多个图表，双击标签可重命名
- **平移 & 缩放** — 鼠标中键拖拽平移，`Ctrl+滚轮` 缩放
- **导出** — 下载 SVG 或 PNG（2 倍分辨率）
- **可折叠面板** — 隐藏/显示编辑器和属性面板
- **自动保存** — 所有标签、位置、颜色、设置自动保存
- **多语言** — 英文、繁体中文、简体中文

## 开始使用

### 从源码运行

```bash
npm install
npm start
```

### 打包为便携式 .exe

```bash
npm run build
```

输出：`dist/Mermaid Editor 1.0.0.exe`

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 渲染图表 |
| `鼠标中键` | 平移画布 |
| `Ctrl+滚轮` | 缩放 |
| `Shift+拖拽节点` | 绘制连线（可视化模式）|
| `Delete` | 删除选取的节点或连线（可视化模式）|
| `Escape` | 取消连线绘制 |
