# Mermaid Editor

**繁體中文** | [简体中文](README.zh-CN.md) | [English](README.md)

桌面版 Mermaid 圖表編輯器，支援視覺化拖拉式編輯、連線繪製、網格吸附、自訂節點顏色。

## 功能

### 程式碼模式
- **Mermaid 渲染** — 輸入 Mermaid 語法即時渲染圖表（`Ctrl+Enter`）
- **拖曳 + 網格吸附** — 自由拖曳節點，自動吸附至可設定的網格
- **自訂顏色** — 點擊節點可自訂背景色、邊框色、文字色
- **邊線跟隨** — 拖曳節點時連接線平滑跟隨移動

### 視覺化模式
- **元素工具箱** — 拖曳流程圖元素（處理、判斷、終端、輸入/輸出、子程序、資料庫）到畫布
- **Shift+拖曳連線** — 按住 Shift 從一個節點拖曳到另一個節點即可建立連線
- **點擊選取連線** — 點擊畫布上的連線即可選取（加寬點擊範圍方便選取）
- **雙擊編輯標籤** — 雙擊連線在點擊位置彈出浮動輸入框編輯標籤
- **Delete 鍵刪除** — 刪除選取的節點或連線
- **雙擊重新命名** — 在畫布上雙擊節點可直接修改名稱
- **即時程式碼同步** — 視覺化操作即時更新 Mermaid 程式碼
- **方向控制** — 切換上→下或左→右佈局
- **節點 & 連線列表** — 左側面板顯示所有節點與連線，支援 inline 編輯和刪除

### 通用功能
- **多分頁** — 同時編輯多個圖表，雙擊分頁可重新命名
- **平移 & 縮放** — 滑鼠中鍵拖曳平移，`Ctrl+滾輪` 縮放
- **匯出** — 下載 SVG 或 PNG（2 倍解析度）
- **可收合面板** — 隱藏/顯示編輯器和屬性面板
- **自動儲存** — 所有分頁、位置、顏色、設定自動保存
- **多語系** — 英文、繁體中文、簡體中文

## 開始使用

### 從原始碼執行

```bash
npm install
npm start
```

### 打包為可攜式 .exe

```bash
npm run build
```

輸出：`dist/Mermaid Editor 1.0.0.exe`

## 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl+Enter` | 渲染圖表 |
| `滑鼠中鍵` | 平移畫布 |
| `Ctrl+滾輪` | 縮放 |
| `Shift+拖曳節點` | 繪製連線（視覺化模式）|
| `Delete` | 刪除選取的節點或連線（視覺化模式）|
| `Escape` | 取消連線繪製 |
